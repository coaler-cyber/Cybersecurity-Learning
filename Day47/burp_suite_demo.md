# Day47: Burp Suite Demo

## Proxy
- Chặn request từ trình duyệt.
- Phân tích header, body.

## Repeater
- Gửi lại request nhiều lần.
- Ví dụ: thử SQL Injection với payload `' OR '1'='1`.

## Intruder
- Tự động brute force mật khẩu.
- Cần cấu hình danh sách payload.

## Scanner (Pro)
- Quét tự động lỗ hổng web.

## Kết luận
Burp Suite là công cụ mạnh mẽ để pentest web, giúp phát hiện SQLi, XSS, CSRF.
